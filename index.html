Como funciona?

O propósito deste site é dar a conhecer ao utilizador o livro mencionado com recurso à interação. Esta é realizada através das indicações dadas pela plataforma. Para melhor perceção do que é esperado do utilizador, decidimos atribuir a cor rosa aos elementos com os quais esperamos que o utilizador interaja. Encorajamos o utilizador a colocar headphones e explorar este livro no computador (desktop).

Para começar, pressione a tecla ENTER.
